📡 API ENDPOINTS (Quick Reference)
================================================================================

GET /api/sweep
  → Run autonomous monitoring
  → Returns: critical_stockouts, risky_suppliers, executive_summary
  → Time: ~5 seconds

POST /api/query
  → Ask AI question
  → Returns: execution_trace, final_answer, confidence, caveats
  → Time: ~3-5 seconds

GET /api/pending-actions
  → List pending approvals
  → Returns: list of {action_id, action_type, details, status}
  → Time: instant

POST /api/pending-actions/:id/status
  → Approve/reject action
  → Returns: {success, new_status}
  → Time: instant

GET /health
  → Health check
  → Returns: {status, timestamp}
  → Time: instant
